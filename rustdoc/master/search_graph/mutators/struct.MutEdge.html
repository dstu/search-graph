pub struct MutEdge<'a, T: Hash + Eq + Clone + 'a, S: 'a, A: 'a> { /* fields omitted */ }

Mutable handle to a graph edge ("edge handle").

This zipper-like type enables traversal of a graph along the edge's source and target vertices.

It enables local graph mutation, whether via mutation of edge data or mutation of graph topology (adding vertices).
